## VaStar

### VaStar es una aplicación realizada de manera formativa con la intención de detectar el camino y la ruta más rápida aplicando los algoritmos como A* de manera heuristica.
### Sin embargo, se ofrece la posibilidad de diseñar de una manera interactiva el entorno para poder comprobar la eficiencia de busqueda en distintas formas.
